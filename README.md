## Description

Game where you go around the map fighting boars and rescuing leaves.

- This project lets users control the fox player sprite and jump around the screen.
- The goal is to rescue all 15/15 leaves
- My edits: Added a debug HUD overlay with live stats, moon gravity toggle, and god mode toggle for playtesting

---

## Interaction Instructions

- Use the up down left right or WASD keys to control the player movements
- Use the space bar to fight boars
- You have 3 lives, R to restart when dead
- T to toggle the debug overlay, M for moon gravity, G for god mode

Example:

> Press T = HUD interface
> Press G = God Mode, no damage taken
> Press M = Moon Gravity mode, jump really high
